## How to run
1. Add `dbconfig.properties` in resources directory. 
2. Add following properties:
```
spring.datasource.username=XXX
spring.datasource.password=XXXXX
```
3. Sync gradle
4. Run project
5. Enter http://localhost:15001 (for dev profile)
